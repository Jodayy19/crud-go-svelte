# Simple CRUD App (Go Fiber + SvelteKit)

## Deskripsi
Aplikasi CRUD sederhana untuk mengelola data user.

## Tech Stack
- Go Fiber (Backend)
- SvelteKit (Frontend)

## Cara Menjalankan

### Backend
cd backend  
go run main.go  

### Frontend
cd frontend  
npm install  
npm run dev  

## API
- GET /users
- POST /users

## Catatan
Data masih disimpan di memory (belum pakai database).
